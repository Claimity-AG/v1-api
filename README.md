# Claimity Partner API notebooks

This folder contains example Jupyter notebooks for the Claimity Partner API.

## Requirements

- Python 3.10+

Install dependencies:

```bash
pip install -r requirements.txt
```

## Configuration

Each notebook reads its configuration from environment variables.

An example `.env` file is provided in [`.env.example`](.env.example:1).

## Notebooks

- Expert API: [`expert_v1.ipynb`](expert_v1.ipynb:1)
- Insurer API: [`insurer_v1.ipynb`](insurer_v1.ipynb:1)
